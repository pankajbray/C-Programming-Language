## 🧭 20-Day Syllabus Roadmap

### 🟢 Week 1: The Fundamentals (Days 1–7)

* **Getting Started:** Constants, Variables, and Data Types.
* **Decision Control:** `if-else`, Nested `if`, and `switch-case`.
* **Loop Control:** `while`, `for`, `do-while`, and break/continue.
* **Case Control:** Complex logical operations.

### 🟡 Week 2: Functions & Memory (Days 8–14)

* **Functions:** Prototypes, Passing values, and Scope.
* **Recursion:** Solving mathematical puzzles (Factorials, Fibonacci).
* **Pointers:** Memory addresses, indirection, and pointer arithmetic.
* **The Preprocessor:** Macros and Header files.

### 🔴 Week 3: Advanced Structures (Days 15–20)

* **Arrays:** 1D/2D arrays and pointers to arrays.
* **Strings:** Manipulation without library functions.
* **Structures:** Defining custom data types and Unions.
* **File Handling:** Read/Write operations and low-level I/O.

## 🧠 Key Learnings

* **Algorithm Thinking:** Breaking down complex problems into small, logical steps.
* **Memory Management:** Understanding how pointers interact with RAM.
* **Efficiency:** Writing code that is both readable and performance-oriented.
* **Debugging:** Using GDB to track variable states and memory leaks.
